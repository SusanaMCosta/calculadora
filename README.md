## Informações

**A calculadora apresenta as 4 operações básicas, além de uma extra:**

* Soma ( + )
* Subtração ( - )
* Multiplicação ( * )
* Divisão ( / )
* Limpar a memória da calculadora ( AC )

## Execução

**Para a execução do projeto, basta seguir os seguintes passos:**

No terminal do seu computador digite:

```sh
yarn install
yarn android
```
ou

```sh
npm install
npm run android
```
